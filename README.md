# Tinder for Dogs
A tinder website made exclusively for dogs where dogs can find their loved ones so that they can be no more lonely in their life.

# About TinDog
A homepage website which provides details of the application to the user like what are the features of the application, its description, how to downnload the application, different plans and subscriptions and testimonials of the clients.

# Tools Used
With the use of HTML, CSS, and Bootstap the whole website has been made. 
The main framework used in building the TinDog website is Bootstrap and its components like Carousel, Card, Buttons, Navbar etc. It also used HTML5, CSS3 and Google Fonts and Font Awesome.
